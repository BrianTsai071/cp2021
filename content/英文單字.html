英文單字

Programming with Python 教科書中收集 2244 個英文單字出現次數, 以及各單字的中英文說明.

從英文教科書中統計各單字出現字數的程式:

先利用 get_pdf_text.py 將 pdf 檔案中的 text 取出.

然後再利用 vocabulary_check.py 查詢各英文單字的中文與英文解釋.

假如導入 pybean.py 與 SQLite 資料庫, 則可以利用 check_dictionary_save_into_sqlite.py 將 10821 個常見的英文單字的中英文說明與例句存入資料庫 (cp2021_textbook_vocabulary_10821.sqlite).

從資料庫中查詢英文單字或列出相關資料:

# 從 pybean.py 導入 Store 與 SQLiteWriter
from pybean import Store, SQLiteWriter

# 設定 library 與 SQLite 資料庫檔案對應
# frozen 決定是否可以動態新增資料表 (table) 中的欄位 (field)
library = Store(SQLiteWriter("cp2021_textbook_vocabulary_10821.sqlite", frozen=False))

# 1 is for all
# 計算資料庫中資料總筆數
count = library.count("vocabulary", "1 order by word")
print(count)

count = 0
# list only num row of data 
# 只列出最前頭 100 筆資料, 英文單字越前面, 表示在 textbook 中出現的次數越多
num =100
# list all data in vocabulary table
for i in library.find("vocabulary","1 limit " + str(num)):
    count += 1
    print (i.word)
    # word 欄位為單字, ec 為英漢,  ee 為英英, es 為例句
    #print (i.word, i.ec, i.ee, i.es)
#print(count)
