教科書導讀

計算機程式課程所採用的 textbook: Programming with Python, 2016 by T R Padmanabhan, 學員可以自行從校園網路中下載.

針對機械設計工程相關的大學課程, 比較建議閱讀英文書, 而這一本 Programming with Python 中的英文單字並不會太多, 建議學員準備一本 B5 紙本筆記專門用來記錄原先不懂的單字, 經過 https://www.oxfordlearnersdictionaries.com/ 或 https://www.merriam-webster.com/ 查詢後, 將英文說明謄寫到紙本筆記中, 隨時翻閱查核是否已經了解這些英文單字的涵意.

當然, 假如您願意接受挑戰, 也可以試著了解一下 https://github.com/mdecourse/lookupdict 這個網際程式 (利用 scrum1 部署在 https://lookupdict.herokuapp.com/, 可惜當時沒有寫成 GET, 否則就可以直接加上 ?word=caste 查前言中所謂的種姓), 其架構概念就如同商用的 https://languages.oup.com/oxford-learners-dictionaries-api/, 可以透過程式對英文的字典進行延伸應用, 例如: 將整本原文書的所有單字取出, 然後利用程式方法進入https://github.com/mdecourse/lookupdict/blob/master/webster_vocabulary.sqlite 或 https://github.com/mdecourse/lookupdict/blob/master/wordnet30.db 資料庫查詢, 並且直接利用 Brython 當作前端, Flask 當作後端, 整理出原文書中所有英文單字的說明圖卡, 然後以每幾秒的緩衝時間, 隨機在網頁中現出特定單字, 讓使用者查核是否已經認得這些原文書中的英文單字意涵, 接著就可安排出現該英文單在在 Webster 英英字典與 Wordnet 中的完整說明, 來強化或印證使用者的記憶.

其實上述說明的待建構網際程式就是與暗記 App 類似的 Flash card 英文學習方法. 有許多為了在短時間強記英文單字的 GRE 或多益應考者, 就是透過這種方法提升自己的英文字彙能力.

至於暗記 (Anki) 最早只使用 Python 在單機上執行, 現在的 https://github.com/ankitects/anki, 已經納入 Rust 與 Typescript, 從這個過程也能發現程式語言的流行會隨著各種工具的出現而發生變化.

總結一下上面這一小段的導讀說明, 就是鼓勵各位學員要多閱讀英文教材, 並且利用程式方法強化學習效果, 假如有衝勁, 可以試著將 https://lookupdict.herokuapp.com/wordnet 與 https://lookupdict.herokuapp.com/ 從現有的原始碼改寫為 GET 或其他延伸應用, 並且學會如何部署到免費的 Heroku.
