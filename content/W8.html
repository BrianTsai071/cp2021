W8

1) 11.11 進入第八週, 下週將取各學員的期中期望成績.

2) Ｗ8 起將導入 GitExtensions, 請直接從 https://github.com/gitextensions/gitextensions/releases/tag/v3.5.4 下載可攜版本後, 放入隨身碟 data 目錄中即可使用.

3) 接下來將要在各分組的伺服器上配置一台 Fossil SCM server (可以提供類似 Github 的網際與分散式版次管理功能), 以便集合各組學員的作業網站與互動討論版.

4) W9 起將要利用 https://mde.tw/cp2021_hw 繼續完成 HW1, HW2 與 HW3 等作業引導 demo, 並利用 Brython 製作網際貪食蛇遊戲的程式碼與教科書導讀來說明各種 Python 程式語法與應用.

5) 在編寫貪食蛇遊戲的過程中, 引導學員了解計算機程式的編寫, 其實就是選擇適合解題的資料結構 (data structures) 與演算法 (algorithms), 選擇一種電腦可資運算 (computation) 與呈現 (presentation) 的程式語言 (即 Brython, CPython, Javascript, C/C++, Typescript, Rust 等), 在電腦硬體與網路串連的架構中 (即 client 與 server 整合應用下), 完成程式設計者所期望的結果.

6) 在手動與自動網際貪食蛇遊戲中, 加入路徑搜尋與機器學習機制, 最後將 Brython 前端程式與 Flask 後端程式結合, 部署在各組的 Fossil SCM server 或 Heroku 雲端伺服器上.
