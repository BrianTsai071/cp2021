抽點

這裡要根據各班學員的學號與 Github 帳號資料, 建立類似 https://mde.tw/cad2021/content/w2task.html 頁面的中的上課學員抽點程式.

假如是在單機上以 CPython 編寫此一抽點程式, 首先必須要先設法讀進學員的學號與 Github 帳號檔案, 因為這個檔案是 1b 在第一週線上課程時所留, 至今可能已經有新的版本資料, 但目前這個程式是希望取出其中的第一行與第二行的資料, 在 1b w4 課堂中所寫的 CPython 程式如下:

with open("cp2021_1b_w1.txt", "r", encoding="utf-8") as f:
    data = f.readlines()

stu = {}

for i in data[1:]:
    data = i.split("\t")
    stu_num = data[0]
    if data[1] == "":
        stu_account = stu_num
    else:
        stu_account = data[1]
        
    stu[stu_num] = stu_account
    #print(stu_num, stu_account)

print(stu["41023237"])

其中包括:

  1. 以 with open() as 讀取檔案內容, 但必須注意檔案內容的編碼, 目前大多使用 utf-8 編碼建立檔案
  2. 以 readlines() 將資料檔案各行的內容存入 list
  3. 以 for 迴圈將包括各種學員回應資料的數列一一取出
  4. 利用數列索引範圍取值的方法, 去掉資料第一列的標題內容
  5. 利用 split() 將採 \t Tab 區隔的資料切成局部數列
  6. 利用局部數列中的索引, 取出索引 0 與索引 1 的學號與 Github 帳號資料
  7. 因應部分學員並未填入 Github 帳號資料, 因此利用 if 判斷式以其學號當作 Github 帳號
  8. 利用 dict 資料結構, 存入各學員的學號與 Github 帳號資料
  9. 最後則利用某一特定學員的學號, 從 dict 取出其 Github 帳號, 進行比對是否正確取值

接下來假如要利用 Brython 讀取已經放入 downloads/online 目錄中的 https://mde.tw/cp2021/downloads/online/cp2021_1b_w1.txt 則可以將上述 CPython 程式中的 with open() as 改為 Brython 的 open(), 隨後的程式碼應該可以重複使用.

採用 Brython 直接在線上編寫的相同功能程式碼如下 (Gist: read_1b_stud_list_into_dict.py):

url = "https://mde.tw/cp2021/downloads/online/cp2021_1b_w1.txt"

data = open(url).readlines()
 
stu = {}
 
for i in data[1:]:
    data = i.split("\t")
    stu_num = data[0]
    if data[1] == "":
        stu_account = stu_num
    else:
        stu_account = data[1]
         
    stu[stu_num] = stu_account
    #print(stu_num, stu_account)
 
print(stu["41023237"])

與 CPython 程式的差異只在:

CPython:

with open("cp2021_1b_w1.txt", "r", encoding="utf-8") as f:
    data = f.readlines()

Brython:

url = "https://mde.tw/cp2021/downloads/online/cp2021_1b_w1.txt"

data = open(url).readlines()

其餘程式碼則完全相同, 但 Brython 程式可以直接採網際方式執行, 就目前此一應用而言較為簡單方便.

根據上述程式, 再加上 random 模組的運用, 可以完成 cp2021 各班的隨機抽點程式如下:

1a 抽點 10 位學員查驗其作業倉儲與網站 (1a 學員倉儲與網站).

1b 抽點 10 位學員查驗其作業倉儲與網站 (1b 學員倉儲與網站).
